This page shows two things for each coin or stock:

  • Trend — a guess at which way the price is leaning right now (up, down, or flat), based only on recent prices. It's a simple statistics calculation, not a prediction from an expert.
  • Accuracy — if you'd trusted that same guess in the past, how often would it have been right? A number near 50% means "no better than a coin flip." Higher doesn't mean reliable going forward — it's a small, recent sample.

Use the Trending up / Trending down buttons above each table to filter the list down to whatever's currently leaning that direction. Nothing here tells you what to buy or sell — it's meant for learning how these signals behave, not for making real decisions.

Stocks

Stock data needs a free API key. If you don't have one yet:

  1. Go to finnhub.io/register and sign up (it's free)
  2. Once logged in, copy the API key shown on your Finnhub dashboard
  3. Paste it into the box below — it stays saved only in this browser, and is never sent anywhere but Finnhub

Finnhub's free tier doesn't offer a market-wide "top movers" feed — that needs a paid plan. Instead, the watchlist below is pre-filled with a default set of well-known large-cap stocks; add your key(s) and scan it. Edit the list to whatever symbols you actually care about.

Trend and accuracy need a second, optional key. Finnhub's free plan doesn't include historical candle data at all (that's paid there), so trend/accuracy instead come from Twelve Data, whose free tier does include daily history. Sign up for a free key at twelvedata.com/register and paste it below too. Without it, price and 24h change still work fine via Finnhub — you'll just see "add Twelve Data key" in the trend/accuracy columns instead. Twelve Data's free tier has a tight per-minute limit, so scans are deliberately slow when this key is present.
